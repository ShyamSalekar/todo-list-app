# To-Do List App 📝

A simple and responsive To-Do List web app built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## 🌟 Features
- Add new tasks
- Mark tasks as complete
- Remove tasks
- Responsive layout
- Works with keyboard (Enter key)

## 🔧 Tech Used
- HTML5
- CSS3 (Flexbox)
- JavaScript (DOM manipulation)

## 🚀 How to Use
1. Clone this repo
2. Open `index.html` in browser
3. Enjoy your tasks ✨
